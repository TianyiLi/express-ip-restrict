## Required:

* nodejs 10.5

or

* docker

## How to use

### With Docker

```
$ docker build -t <tag-name> .
$ docker run -d -p 8080:8080 <tag-name>
$ curl http://localhost:8080/
port expose with 8080
```

or after install nodejs

```
npm install
npm start
```

By test
```
npm install
npm test
```
